# ⚛️ GoBarber Mobile
Projeto mobile desenvolvido no Bootcamp GoStack da Rocketseat.

### Mobile
```
git clone https://github.com/nicolastanski/go-barber-mobile.git
cd go-barber-mobile
yarn install
```

### iOS
```
yarn ios
````

### Android
```
yarn android
````
